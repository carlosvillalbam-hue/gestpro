Flujo: Presupuesto → Proyecto con PO NUEVO
✨ Conversión de Presupuesto Aprobado a Proyecto
1. Presupuesto Se elabora con
desglose de costos
→
2. Cambio de Estado borrador → enviado
→ aprobado
→
3. Botón "Crear Proyecto" Aparece al estar
en estado aprobado
→
4. PO del Cliente Número de PO
+ adjuntar documento
→
5. Proyecto Creado Vinculado al
presupuesto origen
Estados del Presupuesto

📤 Enviado
→
✅ Aprobado
→
📁 Proyecto nueva acción
También: Enviado → ❌ No Aprobado → revisión → Borrador
